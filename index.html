Input reference

  • Use commas, Chinese commas, or line breaks. Escape literal comma separators or colons with a backslash, and backslashes as \\.
  • Use option:3, option:0.5, option:1/365, or option:25% for weights.
  • Escape literal brackets as \[ and \].
  • Choices without weights count as 1.

AI list prompt

GitHub

Help me create a Decisn choice list for this goal: [describe what I am choosing]. Decisn is a local-first random picker that accepts comma-separated or line-separated options, optional weights like option:3, option:25%, option:1/365, and ranges like [1,10]. Suggest practical options and, when useful, recommend simple weights.
